Комплектація

Магнітні насадки для укладання

Насадка для спрямованого потоку:
Створює спрямований потік повітря для надання волоссю більшої гладкості. Об'ємний і контрольований потік повітря виходить із насадки, забезпечує рівномірне висушування.
Концентратор насадки:
Для швидкого укладання волосся. Створює потужний, точно спрямований потік. Допомагає робити будь-яку зачіску із застосуванням гребінця, а також відмінно вигладжує (полірує) волосся.
Дифузор:
Надає ефект природно висушеного волосся (кучеряве, кучеряве), рівномірно розподіляє повітряний потік, висушуючи завитки зі збереженням їхньої природної форми.
Насадка дбайливого висушування:
Швидке і делікатне укладання для користувачів з тонким волоссям і чутливою шкірою голови. Насадка розроблена для дбайливого висушування.
Насадка для неслухняного волосся:
Для створення прямого і гладкого укладання. Приховує пухкі неслухняні волоски під довгими пасмами і надає волоссю ефект гладкого і блискучого волосся.
